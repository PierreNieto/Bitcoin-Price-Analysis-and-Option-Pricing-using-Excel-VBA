# Bitcoin Price Analysis and Option Pricing using Excel VBA

##  Project Overview

This project is an Excel VBA application dedicated to the analysis of Bitcoin (BTC) prices and the valuation of options written on Bitcoin.

The project combines historical data analysis, volatility estimation, and option pricing techniques, fully implemented in **Excel and VBA**.

---

##  Features

- Import and analysis of historical Bitcoin prices  
- Computation of historical volatility  
- Statistical analysis of BTC returns  
- Option pricing on Bitcoin  
- Comparison between Excel formulas and VBA implementations  
- Structured Excel workbook with dedicated sheets for each step

---

##  Workbook Structure

- **Present Values**  
  Calculation and manipulation of present values related to Bitcoin data.

- **BTC Volatility**  
  Historical volatility computation based on Bitcoin price returns.

- **BTC Option Pricing**  
  Pricing of options on Bitcoin using financial models implemented in VBA.

---

##  Tools 

- Historical financial data (Bitcoin prices with stockhistory)
- Financial mathematics (volatility, option pricing with Black Scholes)

---

##  Objectives

- Understand the behavior of Bitcoin prices through historical analysis and the impact of FX on the return.
- Estimate volatility from real market data  
- Implement option pricing models in VBA  
- Strengthen practical skills in Excel VBA applied to finance
